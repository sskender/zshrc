# zshrc

zsh configuration

## Quickstart

```bash
# backup old configuration (optional)
mv ~/.zshrc ~/.zshrc.bkp

# clone repository
cd ~/Documents
git clone https://github.com/sskender/zshrc.git

# link zshrc file
ln -s ~/Documents/zshrc/.zshrc ~/.zshrc

# update
cd ~/Documents/zshrc
git pull
```
